Computational Mechanics (compmech)¶

The compmech package contains semi-analytical routines for linear buckling, linear static and non-linear static analysis of unstiffened cylindrical and conical shells (compmech.conecyl), and panels (compmech.panels). The implementation is based on Python and the low level routines are programmed in Cython.

The details about the theory of the implementations are given in Theory, Implementation and Tutorials.
