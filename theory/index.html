The theory sub-folder of the compmech repository contains the description about the algorithms used in the compmech module. Important details about the implementation are also given.

One could name this part of the repository as the meta-data of the compmech module.

The details about each implemented module are given below:

  • Finite Element Method
  • Meshfree Methods
  • Approximation Functions Used in Various Models
